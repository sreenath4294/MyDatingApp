Login Page

In login page, if the username and password is entered incorrectly, then will show the error message.

The username password validation is done using the Loggedin.java servlet. The code is:


String username = req.getParameter("username");

String password = req.getParameter("password");

try {

stmt = con.createStatement();

rs = stmt.executeQuery("select * from sreenathdatbl1 where username = '"+username+"' and password = '"+password+"'");

if (rs.next()) {

out.println("Hello " + rs.getString("FIRSTNAME") + " " + rs.getString("LASTNAME")+"");

}else {

res.sendRedirect("Loginfailed.html");

}}

catch (Exception e) {

out.println("An error has occurred during the Statement/ResultSet phase. Please check the syntax and study the Exception details!"); }
